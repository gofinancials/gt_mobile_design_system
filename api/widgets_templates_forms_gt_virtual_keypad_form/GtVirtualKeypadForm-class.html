GtVirtualKeypadForm

GtVirtualKeypadForm class

A template widget that displays a full-screen virtual keypad form.

This widget is typically used for PIN entry, passcodes, or amount inputs. It combines a GtDotFormField for visual input representation and a GtKeyPadGrid for user interaction.
