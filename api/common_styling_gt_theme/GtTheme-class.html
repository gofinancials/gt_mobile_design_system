GtTheme

GtTheme class

The foundational blueprint for an application's design system theme.

This abstract class aggregates all styling primitives (colors, typography, spacing, shadows, etc.) and native Flutter theme data. Host applications must implement this class to provide their specific brand identity and token configurations.
