GtPageHeader

GtPageHeader class

Standardized page header: primary screen title with optional subtitle.

Typography is fixed to the design spec:

  • Title: uppercase, 24px, weight 700 (GtTextStyles.h5).
  • Subtitle: 14px, weight 400 (GtTextStyles.bodyS), same color as the title.

Uses GtTextStyles via BuildContext only. Vertical space between title and subtitle uses Column.spacing; when spacingPx is null it falls back to BuildContext.spacingBase (~8dp). Wrap with Padding or padded parents (e.g. SafeArea, ListView padding) as needed.
