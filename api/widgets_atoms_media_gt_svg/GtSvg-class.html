GtSvg

GtSvg class

A widget for efficiently rendering scalable vector graphics (SVGs).

This widget wraps the vector_graphics package for high-performance rendering. It supports two primary modes: as a general-purpose scalable graphic, or as a semantically colored icon via GtSvg.asIcon.
