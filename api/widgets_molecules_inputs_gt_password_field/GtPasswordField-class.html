GtPasswordField

GtPasswordField class

A specialized text input field designed for passwords.

This widget automatically includes a suffix icon to toggle password visibility (obscuring/revealing text), disables auto-correction, sets the appropriate keyboard type, and applies default password validation based on a minimum length.
