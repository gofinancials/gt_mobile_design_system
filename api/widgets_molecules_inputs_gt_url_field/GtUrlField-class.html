GtUrlField

GtUrlField class

A specialized text input field designed specifically for URL addresses.

This widget automatically applies URL validation, sets the appropriate keyboard type, provides autofill hints for URLs, and includes a convenient "paste" suffix button that reads a valid URL directly from the device clipboard.
