Displays a standard bottom sheet containing the given child widget.

The sheet's behavior can be customized with flags such as isDismissable, canDragToClose, and isScrollable. The maxHeightFraction determines the maximum height of the sheet relative to the screen size (defaults to 0.9). Returns a Future that resolves to the value T passed when popping the sheet.
