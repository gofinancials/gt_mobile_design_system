GtRichText

GtRichText class

A versatile rich text widget that parses and renders HTML-like tags using the Go Tech design system.

This widget uses the styled_text package under the hood to apply custom text styles to specific tags within the text string. It comes pre-configured with standard tags like <b>, <i>, <a>, <p>, headings (<h1>-<h6>), and custom tags like <ht> for hashtags and <e> for error styling.

Additionally, it automatically detects hashtags in the raw text and wraps them in <ht> tags.
