GtTextStyles

GtTextStyles class

A centralized styling class that provides access to the application's typographic system and text styles.

Note: This is tentative as there are still inconsistencies in the Figma style guide. These tokens and values may be updated in the future once the design system is fully finalized.

This class uses the provided BuildContext to resolve theme-dependent text styles, scaling factors, and responsive typography tokens as defined in the design system.
