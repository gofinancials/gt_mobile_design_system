GtDecorationImageStyle

GtDecorationImageStyle class

A configuration class that encapsulates styling and behavior parameters for a DecorationImage.

This class acts as a reusable blueprint, allowing you to define image styling rules (like fit, alignment, and filters) once, and easily apply them to different ImageProviders using toDecorationImage.
